## Python Text-Based Game

This project is a text-based game developed in Python for a school assignment.
The game uses functions to manage game flow and player interaction, and
dictionaries and lists to store and retrieve game data.

### Concepts Used
- Python functions
- Dictionaries (key-value data storage)
- Conditional logic
- User input handling
- Program flow control
